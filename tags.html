Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

The most common block elements

Block Elements

include a paragraph of text
Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
ol Ordered list Introduces a numbered list
ul Unordered list Introduces a bulleted list

The most common inline elements

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> Makes text stand out strongly (mostly bold)
a anchor make hyperlink <a href="http://learnenough.com/">Learn Enough</a> Adds a clickable hyperlink that takes you to another page.
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl"> Allows you embed or link to image files.
code code Sets code samples off from the surrounding text <code>Sets code samples in a monospaced font </code> Sets code samples in monospaced font
li List item An item in a numbered or unordered list <ol><li>This is a list item.</li></ol>
  1. This is a list item.
